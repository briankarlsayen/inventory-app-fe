## Features

- User authentication
- Role based access control
- REST API for managing stocks and orders

## Tech Stacks

- **Framework**: VueJS(frontend), Django(backend)
- **Database**: MongoDB
- **State Management**: Pinia
- **Styling**: Tailwindcss v4

## Installation

### Clone repository

### Install dependencies

```
npm install
```

### Copy .env file

```
cp .env.example .env
```

#### Start application

```
npm run dev
```
